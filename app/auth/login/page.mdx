# การเข้าสู่ระบบ (Login)

หลังจากที่คุณมีบัญชีแล้ว หรือเพิ่งสมัครสมาชิกเสร็จสิ้น คุณสามารถเข้าสู่ระบบได้ดังนี้:

1.  **ไปยังหน้าเข้าสู่ระบบ**:
    - โดยทั่วไป คุณจะเข้าถึงหน้านี้หลังจากสมัครสมาชิกสำเร็จ หรือเข้าผ่าน URL ที่ระบบกำหนดให้โดยตรง
2.  **กรอกข้อมูลในฟอร์ม "เข้าสู่ระบบ"**:
    - **อีเมล (Email)**: ป้อนที่อยู่อีเมลที่คุณได้ลงทะเบียนไว้ในช่อง "อีเมล" (ตัวอย่าง: `m@example.com`)
    - **รหัสผ่าน (Password)**: ป้อนรหัสผ่านของบัญชีคุณในช่อง "รหัสผ่าน"
3.  **คลิกปุ่ม "เข้าสู่ระบบ" (Login)**:
    - หลังจากกรอกข้อมูลครบถ้วนแล้ว ให้คลิกที่ปุ่ม "เข้าสู่ระบบ"
    - หากอีเมลหรือรหัสผ่านไม่ถูกต้อง ระบบจะแสดงข้อความแจ้งเตือน ให้คุณตรวจสอบข้อมูลและลองอีกครั้ง
4.  **เมื่อเข้าสู่ระบบสำเร็จ**:
    - ระบบจะนำคุณไปยังส่วนถัดไป ซึ่งอาจเป็นหน้า "ตั้งค่าโปรไฟล์" (หากคุณยังไม่เคยตั้งค่า) หน้า "ตั้งค่าบริษัท" (หากโปรไฟล์เรียบร้อยแต่ยังไม่มีบริษัท) หรือหน้า "แดชบอร์ด" (Dashboard) หลักของโปรแกรม หากทุกอย่างตั้งค่าเรียบร้อยแล้ว

> [!TIP]
>
> - หากคุณยังไม่มีบัญชีผู้ใช้ คุณสามารถคลิกที่ลิงก์ "**สมัครสมาชิก**" (Register) บริเวณด้านล่างของฟอร์มเพื่อไปยังหน้าสมัครสมาชิกได้ทันที
