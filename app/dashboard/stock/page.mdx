# แท็บสต็อก (Stock)

แท็บนี้จะให้ข้อมูลเชิงลึกเกี่ยวกับสต็อกสินค้าของคุณโดยเฉพาะ

## 1 แผนภูมิสต็อกตามกลุ่ม (Stock by Group Chart)

*   **ประเภทแผนภูมิ:** แผนภูมิแท่ง (Bar Chart)
*   **แสดงอะไร:** การกระจายของสต็อกสินค้าตามกลุ่มสินค้าต่างๆ โดยแสดงทั้งจำนวนรวม (Total Count) และมูลค่ารวม (Total Value) ของสินค้าในแต่ละกลุ่ม
    *   แกน X (แนวนอน) คือชื่อกลุ่มสินค้า
    *   แกน Y ซ้าย (แนวตั้ง) คือจำนวนรวมต่อกลุ่ม (Total Count per Group)
    *   แกน Y ขวา (แนวตั้ง) คือมูลค่ารวมต่อกลุ่ม (Total Value per Group)
*   **คำอธิบาย:** "Distribution of stock across groups" (การกระจายของสต็อกตามกลุ่มต่างๆ)
*   **วิธีใช้:**
    *   เปรียบเทียบแท่งกราฟของแต่ละกลุ่มเพื่อดูว่ากลุ่มสินค้าใดมีจำนวนหรือมูลค่ามากที่สุด
    *   แท่งกราฟแต่ละกลุ่มจะมีสองสี: สีหนึ่งสำหรับจำนวน (Count) และอีกสีสำหรับมูลค่า (Value) ดูคำอธิบาย (Legend) เพื่อทราบว่าสีใดแทนอะไร
    *   เลื่อนเมาส์ไปบนแท่งกราฟเพื่อดูจำนวนและมูลค่าที่แน่นอนของแต่ละกลุ่ม

## 2 ตารางข้อมูลสต็อกยอดนิยม (Top Stocks Tables)

1.  **Top Stocks by Value (สต็อกยอดนิยมตามมูลค่า):**
    *   **แสดงอะไร:** รายการสินค้า 5 อันดับแรกที่มีมูลค่ารวมสูงสุดในคลัง (คำนวณจาก จำนวน x ราคาต่อหน่วย)
    *   **คำอธิบาย:** "Highest value items in inventory" (รายการสินค้าที่มีมูลค่าสูงสุดในคลัง)
    *   **คอลัมน์ในตาราง:**
        *   **Name (ชื่อ):** ชื่อสินค้า
        *   **Count (จำนวน):** จำนวนสินค้าที่มีอยู่
        *   **Unit Price (ราคาต่อหน่วย):** ราคาของสินค้าต่อหนึ่งหน่วย
        *   **Total Value (มูลค่ารวม):** มูลค่ารวมของสินค้ารายการนั้น (จำนวน x ราคาต่อหน่วย)
    *   **วิธีใช้:** ระบุสินค้ารายการสำคัญที่มีมูลค่าสูง เพื่อให้ความสำคัญกับการจัดการสินค้ารายการเหล่านี้เป็นพิเศษ

2.  **Top Stocks by Count (สต็อกยอดนิยมตามจำนวน):**
    *   **แสดงอะไร:** รายการสินค้า 5 อันดับแรกที่มีจำนวนชิ้นมากที่สุดในคลัง
    *   **คำอธิบาย:** "Most numerous items in inventory" (รายการสินค้าที่มีจำนวนมากที่สุดในคลัง)
    *   **คอลัมน์ในตาราง:**
        *   **Name (ชื่อ):** ชื่อสินค้า
        *   **Count (จำนวน):** จำนวนสินค้าที่มีอยู่
        *   **Unit Price (ราคาต่อหน่วย):** ราคาของสินค้าต่อหนึ่งหน่วย
        *   **Description (รายละเอียด):** คำอธิบายเพิ่มเติมเกี่ยวกับสินค้า (หากมี)
    *   **วิธีใช้:** ระบุสินค้าที่มีปริมาณหมุนเวียนสูง หรือสินค้าที่อาจต้องการพื้นที่จัดเก็บมาก

## 3 ตารางแจ้งเตือนสต็อก (Stock Alert Tables)

1.  **Low Stock Items (รายการสต็อกเหลือน้อย):**
    *   **แสดงอะไร:** รายการสินค้าที่จำนวนคงเหลือต่ำกว่าเกณฑ์ที่กำหนด (หรือใกล้จะหมด)
    *   **คำอธิบาย:** "Items that need replenishment" (รายการสินค้าที่ต้องเติมสต็อก)
    *   **ไอคอนหัวข้อ:** รูปวงกลมมีเครื่องหมายตกใจสีเหลือง (AlertCircle)
    *   **คอลัมน์ในตาราง:**
        *   **Name (ชื่อ):** ชื่อสินค้า
        *   **Current Count (จำนวนปัจจุบัน):** จำนวนสินค้าที่เหลืออยู่
        *   **Status (สถานะ):** แสดงแถบความคืบหน้า (Progress Bar) ที่บ่งบอกระดับสต็อกที่เหลือ และคำว่า "Low" (น้อย) สีเหลือง
    *   **วิธีใช้:** ตรวจสอบรายการนี้เป็นประจำเพื่อวางแผนการสั่งซื้อสินค้าเพิ่มเติม ป้องกันปัญหาสินค้าหมดสต็อก

2.  **Inactive Stock (สต็อกที่ไม่มีการเคลื่อนไหว):**
    *   **แสดงอะไร:** รายการสินค้าที่ไม่มีการเคลื่อนไหว (ไม่มีธุรกรรมเข้าหรือออก) เป็นระยะเวลานาน
    *   **คำอธิบาย:** "Items with no recent activity" (รายการสินค้าที่ไม่มีกิจกรรมล่าสุด)
    *   **ไอคอนหัวข้อ:** รูปนาฬิกา (Clock)
    *   **คอลัมน์ในตาราง:**
        *   **Name (ชื่อ):** ชื่อสินค้า
        *   **Count (จำนวน):** จำนวนสินค้าที่มีอยู่
        *   **Last Updated (อัปเดตล่าสุด):** วันที่และเวลาที่มีการอัปเดตข้อมูลสินค้ารายการนี้ครั้งล่าสุด แสดงในรูปแบบ "X [หน่วยเวลา] ago"
    *   **วิธีใช้:** พิจารณาสินค้าในรายการนี้ อาจเป็นสินค้าขายช้า หรือสินค้าที่ตกรุ่น เพื่อวางแผนการจัดการสต็อก เช่น จัดโปรโมชั่นระบายสินค้า
