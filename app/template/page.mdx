# เทมเพลต 

สร้างและใช้เทมเพลตเพื่อทำธุรกรรมซ้ำได้ง่ายขึ้น ฟีเจอร์นี้ช่วยให้คุณสามารถจัดการสินค้าที่มีการเบิกจ่ายหรือนำเข้าเป็นประจำได้อย่างรวดเร็วและมีประสิทธิภาพ

> [!TIP] เทมเพลตช่วยประหยัดเวลาในการจัดการธุรกรรมที่เกิดขึ้นบ่อยๆ เช่น การเบิกวัตถุดิบประจำวัน การส่งสินค้าให้ลูกค้าประจำ หรือการนำเข้าสินค้าจากซัพพลายเออร์ประจำ

## การสร้างเทมเพลต

สร้างเทมเพลตใหม่ได้ด้วยขั้นตอนง่ายๆ ดังนี้:

1. คลิก "สร้างแบบฟอร์มใหม่"
2. เลือกสินค้าจากคลัง (ส่วนซ้ายของหน้าจอ)
   - ค้นหาสินค้าด้วยชื่อ รหัส หรือกลุ่ม
   - คลิกที่สินค้าเพื่อเพิ่มในเทมเพลต
   - กำหนดจำนวนสำหรับแต่ละรายการ
3. กรอกข้อมูลเทมเพลต
   - ชื่อเทมเพลต (จำเป็น)
   - คำอธิบาย (ไม่บังคับ)
   - ตั้งค่าเป็นส่วนตัว/สาธารณะ
4. คลิก "สร้างเทมเพลต"

## การใช้และจัดการเทมเพลต

เมื่อสร้างเทมเพลตแล้ว คุณสามารถใช้งานและจัดการได้ดังนี้:

### การใช้เทมเพลต
1. ไปที่หน้า Templates
2. ค้นหาเทมเพลตที่ต้องการใช้
3. คลิก "Use Template"
4. ตรวจสอบและปรับแก้รายการถ้าจำเป็น
5. เลือกประเภทธุรกรรม (นำเข้า/นำออก)
6. คลิก "สร้างธุรกรรม"

### การจัดการเทมเพลต
คุณสามารถจัดการเทมเพลตของคุณในแท็บ "Private Templates":

- **แก้ไข**: คลิก "Edit" เพื่อปรับปรุงรายการหรือข้อมูลเทมเพลต
- **ลบ**: คลิก "Delete" เพื่อลบเทมเพลตที่ไม่ต้องการใช้อีกต่อไป
- **คัดลอก**: คลิก "Duplicate" เพื่อสร้างสำเนาของเทมเพลต

## ประเภทของเทมเพลต

CopaStock รองรับเทมเพลตสองประเภท:

1. **เทมเพลตส่วนตัว (Private Templates)**: มองเห็นและใช้งานได้เฉพาะคุณเท่านั้น เหมาะสำหรับธุรกรรมเฉพาะบุคคล
2. **เทมเพลตสาธารณะ (Public Templates)**: มองเห็นและใช้งานได้โดยทุกคนในองค์กร เหมาะสำหรับธุรกรรมที่ใช้ร่วมกัน
